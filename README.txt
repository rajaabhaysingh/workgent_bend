Steps to run backend_sample:
---------------------------------------------

1. Open backend_sample folder in terminal
2. Create a virtual environment and activate it (optional but recommended)
3. run the command "pip install -r requirements.txt"
4. run "python3 manage.py runserver"
5. Done! Open "http://localhost:8000/swagger/" or "http://localhost:8000/redoc/" in your web browser.
